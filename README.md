# Statistical Analysis in Microsoft Excel
Risk-Based Part Failure Probability for Maintenance Planning Dashboard created in Microsoft Excel
This project includes a dataset with 1000 observations and statistical analyses such as hypothesis testing, regression analysis, descriptive statistics, and data visualization.
